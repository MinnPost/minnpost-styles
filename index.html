MinnPost Styles

The MinnPost super-fly style guide, specifically focused on interactive and news applications made by the MinnData team.

Basic usage

Since these styles were made for projects that will get embedded in existing sites, all styles are prefixed with .minnpost-styles. This means you should wrap your markup like so:

<div class="minnpost-styles">
  ...
</div>

Or, if you are not embedding, simply add the class to the body:

<body class="minnpost-styles">
...
</body>

Links

Links are used for inline navigation. For external links, it is often a good idea to use the target="_blank" attribute as it will open the link in a new tab.

This is a link.

<a href="#">link</a>
